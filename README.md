# Accident-Prone

A bladder / bowel / hydration roleplay plugin for **Paper / Spigot 1.21.4** Minecraft servers, with layered absorbency, accidents, caregivers, and a custom HUD driven by a bundled resource pack.

## 📖 [Read the docs](https://storynook.github.io/Accident-Prone-1.21.4/)

Full reference docs — install guide, every subsystem, admin setup — live on the published site.

## Build from source

```bash
mvn package
```

Output: `target/Accident-Prone-<version>.jar`. Drop into your server's `plugins/` folder.
