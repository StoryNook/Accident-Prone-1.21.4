## What it adds

- A two-stat urgency loop (bladder / bowels) that fills over time and triggers warnings.
- Four-tier layered absorbency (nothing → underwear → pull-up → diaper → thick) with visible wet/messy states.
- A configurable accident system, hypnosis, caregivers, and rash mechanics — all admin-gated.
- A custom action-bar / sidebar HUD driven by a companion resource pack (bundled with the plugin).
